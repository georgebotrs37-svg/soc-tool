أداة بسيطة ومنظمة لبناء مركز عمليات أمنية (SOC) باستخدام لغة Go. تتبع هذه الأداة هيكلية برمجية احترافية تسمح بتوسيع نطاق المراقبة والتحليل الأمني.

## 📌 مميزات المشروع (Features)
- **Modular Architecture**: تقسيم الكود إلى حزم منفصلة (API, Database, Ingestion).
- **Log Ingestion Engine**: محرك جاهز لاستقبال وسحب السجلات (Logs) ومعالجتها.
- **RESTful API**: واجهة برمجية لعرض الأحداث الأمنية والتفاعل مع الأداة.
- **Fast & Lightweight**: مبني بالكامل بلغة Go لضمان السرعة وكفاءة استهلاك الموارد.

## 📂 هيكل المشروع (Project Structure)
```text
soc-tool/
├── internal/
│   ├── api/          # إدارة سيرفر الويب والواجهات البرمجية
│   ├── database/     # تهيئة وإدارة الاتصال بقواعد البيانات
│   └── ingestion/    # محرك سحب وتحليل السجلات (Logs)
├── main.go           # نقطة الانطلاق الرئيسية للتطبيق
└── go.mod            # ملف إدارة المكتبات والاعتمادات
🛠️ المتطلبات (Prerequisites)
لغة البرمجية Go (إصدار 1.20 أو أحدث).
أي متصفح ويب لتجربة الـ API.

تشغيل الأداة:
